# User Stories

1. As a support agent, I want to ask whether an unopened item can be refunded within 14 days and receive a cited answer.
2. As a support agent, I want to retrieve the latest order status instead of trusting model memory.
3. As a support agent, I want the assistant to draft a refund proposal instead of directly issuing a refund.
4. As a support lead, I want risky requests to be escalated to human review.
5. As a support lead, I want a short structured summary attached to each escalated ticket.
6. As a developer, I want to inspect tool traces when a case fails.
7. As a developer, I want a fixed eval set that covers FAQ, lookup, action requests, and handoff.
